# IPL 2026 PowerPlay Analysis Dashboard

An end-to-end Data Analytics project focused on analyzing IPL 2026 team and player performance. Data was collected from IPL Stat Center and structured into separate batting, bowling, auction, and match result datasets. Using Python in Jupyter Notebook, the data was cleaned, transformed, and analyzed through Exploratory Data Analysis (EDA). MySQL was used to perform PowerPlay-specific analysis, while interactive Power BI dashboards were developed to visualize key insights, including team performance, player statistics, auction spending trends, and tournament highlights.

### Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- MySQL
- Power BI

### Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- PowerPlay Performance Analysis
- Team & Player Comparison
- Auction Spending Analysis
- Interactive Power BI Dashboards
